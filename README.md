# Agriculture Production Prediction

## Project Overview
This project predicts agricultural production using Machine Learning. The dataset was analyzed, cleaned, and visualized to understand production trends. A Linear Regression model was used to predict agricultural production.

## Dataset
Source: Kaggle

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Import Dataset
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Train-Test Split
6. Linear Regression Model
7. Model Evaluation
8. Prediction

## Results
- Successfully trained a Linear Regression model.
- Visualized production trends.
- Compared actual and predicted values using graphs.
